# JustHex

This is a simple program to view and edit files in "hex" form.

## Manual

To use this program, simply open the "index.html" in your web browser.
In hex collumn, you can only write the base16 symbols (0-9 and A-F), while in string collum, you can use most of your keyboard symbols


You can change some settings like row and byte-group size in the menu at the top. 
They will be saved to your browser's storage so you wouldn't have to edit them each time.
